# 📊 Logistic Regression Implementation

A simple implementation of Logistic Regression to understand the fundamentals of binary classification, probability prediction, and model evaluation.

## 🚀 Features

- Logistic Regression model training
- Binary Classification
- Data preprocessing
- Model evaluation
- Prediction and accuracy analysis

## 🛠 Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

## ▶️ Run the Project

```bash
git clone https://github.com/abhinav29890/ML_LogisticRegression.git

cd ML_LogisticRegression

python main.py
```

## 📈 Learning Outcomes

- Understanding Logistic Regression
- Binary Classification
- Sigmoid Function
- Model Training & Prediction
